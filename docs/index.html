Highlights

  • Palm-first parser: open WQVLink/WQVColor DBs, CASIJPG PDB files, raw dumps, or standalone PNG/JPEG images while preserving capture metadata.
  • Colour stitching: pair WQVColorDB with CASIJPG companions to display original colour captures when available.
  • Dual-pane GUI: thumbnails, metadata panel, and original/upscaled previews with responsive background workers.
  • Configurable upscaling pipelines: combine Pillow resamplers with Real-ESRGAN or NeoSR models and save named presets.
  • Trainer: generate degradations, train RRDB/NeoSR models and export deployable checkpoints.

Quick Start

  1. Install requirements (Python 3.9+). From the project root: 
    python -m pip install -e .[dev]
  2. Run the viewer: 
    python -m wqv_viewer

Models & Training

Bundled Real-ESRGAN and repository custom models (e.g. models/custom/wqv_neosr_x4.pth) are auto-discovered. Drop additional .pth files into models/custom and restart to pick them up. Use the trainer to warm-start from RRDB checkpoints.
